# L&L Locadora de Carros

Projeto de sistema para locação de veículos desenvolvido com **Django**. O sistema permite visualizar categorias de carros e realizar simulações de aluguel.

## 🛠️ Tecnologias Utilizadas
- **Python 3.13**
- **Django Framework**
- **HTML5 & CSS3** 
- **JavaScript**
